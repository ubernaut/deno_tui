# Rebranding Notes

> Status: exploratory note only. This is not a TODO item, an approved rename, or a commitment to change package names,
> APIs, repositories, or existing Neon Exodus terminology.

The project has grown substantially beyond the scope of the library from which it was originally forked. A future
rebrand could reflect its broader identity: advanced terminal UI infrastructure, Unicode and glyph rendering, truecolor
and 3D ASCII visualization, window and workspace management, multiple renderers, application services, and the larger
Neon Exodus aesthetic.

## Candidate Names

1. **Neon Exodus** — a cinematic and memorable umbrella brand.
2. **Exodus UI** — a clean, direct framework identity.
3. **ExoTUI** — compact and immediately recognizable as a TUI project.
4. **Neon Ark** — carrying terminal computing into a new era.
5. **Exodus Forge** — oriented around building complete applications.
6. **Neon Foundry** — industrial, extensible, and framework-oriented.
7. **Hyperglyph** — suited to advanced text, Unicode, glyph, and 3D ASCII work.
8. **Phosphor Ark** — terminal heritage paired with a science-fiction identity.
9. **Lumenforge** — a polished, powerful, renderer-neutral framework name.
10. **Neonframe** — emphasizes windowing, layout, and rendering.
11. **Glyph Exodus** — connects directly to the project's text-rendering evolution.
12. **Chromaforge** — emphasizes truecolor rendering and creation.
13. **PrismTTY** — colorful, technical, and compact.
14. **NovaCell** — terminal cells rebuilt into a much larger system.
15. **Radiant Grid** — suited to cells, layouts, dashboards, and windows.
16. **Spectral Forge** — atmospheric and ambitious.
17. **Escape Velocity UI** — references both terminal escape codes and Exodus.
18. **Afterglow** — an elegant standalone name with neon associations.
19. **LuxGrid** — a short, modern name centered on illuminated cell layouts.
20. **Neon Dominion** — a deliberately grand identity for an ambitious framework.

## Early Shortlist

- **Neon Exodus** for the complete project or ecosystem.
- **Hyperglyph** for the core library or rendering engine.
- **Lumenforge** for a professional framework identity.
- **Phosphor Ark** for a distinctive terminal-native identity.
- **ExoTUI** when immediate technical clarity matters most.

One possible brand architecture would use **Neon Exodus** for the overall ecosystem, **Hyperglyph** for its rendering
core, and **Exodus Forge** for the application workbench.

These names have not been screened for trademarks, package registries, domain availability, search ambiguity, or
conflicts with existing projects. Any future decision should perform that research before changing public identifiers.
